Social Media Campaign Performance Tracker


Project Overview
This project focuses on analyzing Facebook Ad Campaign data to evaluate marketing performance. By tracking key metrics, this dashboard provides insights into audience behavior, cost efficiency, and the overall effectiveness of different advertising segments.

Tools & Technologies
Power BI Desktop: Dashboarding and Interactive Visualization.
DAX (Data Analysis Expressions): Calculated measures for CTR, CPC, and ROI.
Data Source: Facebook Ad Campaign Dataset (data.csv).

Visuals & Key Insights
Executive Summary: KPI cards showing Total Spend, Total Conversions, and Average ROI.
CTR Analysis: Line charts showing how engagement fluctuates over the reporting period.
Demographic Deep-Dive: Bar and Donut charts analyzing performance by **Age** and **Gender**.
Efficiency Matrix: A scatter plot comparing Ad Spend against Approved Conversions to identify the best-performing segments.
Interactive Slicers: Filters for Gender, Age Group, and Date for granular analysis.
